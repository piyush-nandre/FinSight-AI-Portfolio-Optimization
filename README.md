# FinSight AI

## Overview

FinSight AI is an AI-powered portfolio optimization platform designed for Indian investors.

The platform combines Modern Portfolio Theory (MPT), Efficient Frontier analysis, Stress Testing, Retrieval-Augmented Generation (RAG), FAISS Vector Search, and Google Gemini AI to help users build and evaluate investment portfolios.

---

## Features

### Portfolio Optimization

* Modern Portfolio Theory (MPT)
* Expected Returns Calculation
* Covariance Matrix Estimation
* Efficient Frontier Visualization
* Risk-Based Portfolio Construction

### Risk Profiles

* Conservative (Minimum Volatility)
* Moderate (Maximum Sharpe Ratio)
* Aggressive (Maximum Quadratic Utility)

### Portfolio Analytics

* Portfolio Performance Metrics
* Investment Allocation Breakdown
* Portfolio Composition Visualization
* Portfolio Health Score

### Stress Testing

* RBI Rate Hike Scenario
* IT Slowdown Scenario
* Market Correction Scenario
* Bull Market Scenario
* China Slowdown
* AI Boom
* Recession
* Oil / crude price hike

### AI-Powered Insights

* Financial News Retrieval
* Article Processing
* Embedding Generation
* FAISS Vector Search
* Retrieval-Augmented Generation (RAG)
* Google Gemini Integration

### Recommendation Engine

* Portfolio Health Score
* Risk Assessment
* Investment Recommendation
* Strengths & Risks Analysis

## Installation

Clone the repository:

```bash
git clone <repository_url>
cd FinSight-AI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create Streamlit secrets:

```toml
GEMINI_API_KEY="YOUR_API_KEY"
```

Run the application:

```bash
streamlit run app.py
```
